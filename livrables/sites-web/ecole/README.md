# ecole/

Livrables pour **l'Apreneur Académie** (formations, cours, programmes).

**Ce qui va ici :**
- Plans de formation et curricula
- Contenus de modules (leçons, exercices, quiz)
- Emails de séquence onboarding / nurturing
- Pages de vente de formation
- Supports pédagogiques (fiches, récaps, workbooks)
- Calendriers de cohorte

**Convention de nommage :**
`AAAA-MM-JJ_[formation-X]_[module-N|email|page-vente]_v1.md`

**Exemple :**
`2026-06-04_formation-ia-freelance_module2-lecon3.md`
